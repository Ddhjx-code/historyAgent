## 三、项目核心：提示词生成

### 3.1 为什么提示词生成是关键？

1. **图像生成质量取决于提示词**：Seedream 需要精确的视觉描述才能生成高质量图片
2. **视频动态效果取决于提示词**：Seedance 需要清晰的镜头运动描述
3. **一致性控制**：多场景需要保持人物、风格、色调一致
4. **自动化程度**：提示词质量决定了自动化流程的可行性

### 3.2 两类提示词

#### 类型一：图像提示词 (Image Prompt)
用于 Seedream 生成静态关键帧图片

**核心要素**：
- 视觉风格（历史写实 / 3D CG / 纪录片风格）
- 场景描述（地点、环境、天气、光线）
- 人物描述（外貌、服饰、姿态、表情）
- 构图方式（景别、视角、焦点）
- 色调氛围（暖色调、冷色调、戏剧光效）

**示例**：
```
历史写实风格，古罗马帝国宫廷，公元138年。
哈德良皇帝身着紫色长袍，头戴橄榄枝冠冕，坐在大理石宝座上。
他面前站着安敦宁·毕尤，身穿白色托加长袍，恭敬地接受收养。
宫廷内光线从高窗射入，形成戏剧性的光影效果。
大理石柱廊、马赛克地板、罗马鹰旗在背景中若隐若现。
电影质感，16:9 横屏，暖色调，史诗感。
```

#### 类型二：动态提示词 (Motion Prompt)
用于 Seedance 生成视频片段

**核心要素**：
- 镜头运动（推进/拉远/环绕/跟随/固定）
- 主体动作（人物行为、物体移动）
- 环境变化（光线变化、天气变化）
- 节奏控制（缓慢/快速/加速/减速）
- 转场效果（淡入淡出/切换/溶解）

**示例**：
```
镜头从宫廷大门缓慢推进，穿过罗马柱廊。
哈德良皇帝从宝座上缓缓站起，走向安敦宁·毕尤。
阳光从高窗移动，光影在地面缓缓移动。
皇帝伸出手，轻轻搭在安敦宁肩膀上。
镜头缓缓上移，展现宫廷穹顶的壁画。
节奏缓慢庄重，符合历史仪式感。
```

## 四、提示词生成方法论

### 4.1 从口播稿到场景拆解

**输入**：口播稿文本
**输出**：场景列表 + 时间轴

```markdown
## 场景拆解示例

### 场景1：罗马帝国权力交接
- 时间：0:15-0:45
- 核心事件：哈德良收养安敦宁·毕尤
- 画面需求：罗马宫廷、皇帝、仪式感
- 情绪基调：庄严、历史感
- 旁白：【事件一】公元138年2月25日...

### 场景2：玄奘归来
- 时间：2:30-3:30
- 核心事件：玄奘携佛经回到长安
- 画面需求：长安城门、马队、佛经箱笼
- 情绪基调：喜悦、庄严
- 旁白：【事件三】公元645年2月25日...
```

### 4.2 从场景到图像提示词

**转换规则**：

| 场景要素 | 图像提示词要素 |
|---------|---------------|
| 时间年代 | 服饰风格、建筑风格 |
| 地点 | 场景环境、地理特征 |
| 人物 | 外貌、服饰、姿态 |
| 事件 | 动作瞬间、关键画面 |
| 情绪 | 光线、色调、构图 |

### 4.3 从场景到动态提示词

**转换规则**：

| 场景要素 | 动态提示词要素 |
|---------|---------------|
| 叙事节奏 | 镜头运动速度 |
| 空间关系 | 镜头调度（推进/拉远/环绕） |
| 人物行为 | 动作描述 |
| 环境变化 | 光影变化、天气变化 |
| 情绪渲染 | 运镜节奏、景别变化 |

## 六、实现步骤

### Step 1：场景解析器
- 解析口播稿结构
- 提取【事件】、【画面】、【旁白】
- 生成场景列表和时间轴

### Step 2：提示词生成器（核心）
- 基于场景描述生成图像提示词
- 基于场景描述生成动态提示词
- 保持多场景一致性（人物、风格、色调）

### Step 3：图片生成器
- 调用 Seedream API
- 生成关键帧图片
- 保存到 outputs/frames/

### Step 4：视频生成器
- 调用 Seedance API
- 基于图片 + 动态提示词生成视频
- 保存到 outputs/videos/

### Step 5：视频合成器
- 拼接视频片段
- 添加音频（TTS 旁白）
- 添加字幕
- 输出最终成片

## 七、API 调用示例

### 7.1 图像生成 (Seedream)

```python
from openai import OpenAI
import os

client = OpenAI(
    base_url="https://ark.cn-beijing.volces.com/api/v3",
    api_key=os.getenv("VOLCENGINE_API_KEY"),
)

def generate_image(prompt: str, size: str = "1920x1080"):
    """生成关键帧图片"""
    response = client.images.generate(
        model="doubao-seedream-4.5",
        prompt=prompt,
        size=size,
        n=1,
    )
    return response.data[0].url
```

### 7.2 视频生成 (Seedance)

```python
import requests
import json

def generate_video(image_url: str, motion_prompt: str, duration: int = 5):
    """图生视频"""
    response = requests.post(
        "https://ark.cn-beijing.volces.com/api/v3/contents/generation",
        headers={
            "Authorization": f"Bearer {os.getenv('VOLCENGINE_API_KEY')}",
            "Content-Type": "application/json"
        },
        json={
            "model": "doubao-seedance-2.0",
            "content": [
                {"type": "image_url", "image_url": {"url": image_url}},
                {"type": "text", "text": motion_prompt}
            ],
            "video_config": {
                "duration": duration,
                "resolution": "1080p",
                "aspect_ratio": "16:9"
            }
        }
    )
    return response.json()
```
