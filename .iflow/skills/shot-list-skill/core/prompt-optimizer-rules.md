# Prompt优化器规则

## 概述

本文档定义了Prompt优化规则，用于优化生成的图像prompt，提高prompt的简洁性、流畅性、专业性和准确性。

## 优化目标

1. **简洁性**：删除冗余描述，保留核心视觉特征
2. **流畅性**：确保描述逻辑连贯，增强画面感
3. **专业性**：优化摄影术语和技术参数
4. **准确性**：修正历史错误，确保文化尊重

## 优化规则

### 规则1：简洁性优化

#### 1.1 删除冗余描述

**问题**：重复描述同一元素，增加prompt长度

**优化方法**：
1. 识别重复的描述
2. 保留最精确的描述
3. 删除冗余的描述

**示例**：

**优化前**：
```
40岁的林江迈，一位40岁的台湾寡妇，脸上写满疲惫，脸上写满对生活的疲惫，眼神中有对明天的担忧，眼神中充满担忧。
```

**优化后**：
```
40岁的林江迈，一位台湾寡妇，脸上写满疲惫，眼神中有对明天的担忧。
```

#### 1.2 合并相似元素

**问题**：相似元素分散描述，缺乏整合

**优化方法**：
1. 识别相似的元素
2. 使用简洁的描述整合
3. 保留核心特征

**示例**：

**优化前**：
```
破旧的木结构建筑，陈旧的木结构建筑，斜坡屋顶，日式装饰，日式招牌。
```

**优化后**：
```
破旧的木结构建筑，斜坡屋顶，日式装饰和招牌。
```

#### 1.3 保留核心视觉特征

**问题**：包含过多次要细节，分散视觉焦点

**优化方法**：
1. 识别核心视觉特征（人物、动作、环境）
2. 保留核心特征
3. 删除次要细节

**示例**：

**优化前**：
```
40岁的林江迈，一位台湾寡妇，身穿打补丁的粗布衣裳，脸上写满疲惫，眼神中有对明天的担忧，蹲在路边摆摊，身体前倾，正在整理香烟摊位，旁边散落着几个空盒子，远处有几个行人走过，街道尽头有模糊的建筑轮廓。
```

**优化后**：
```
40岁的林江迈，身穿打补丁的粗布衣裳，脸上写满疲惫，蹲在路边摆摊，正在整理香烟摊位。
```

### 规则2：叙事流畅性优化

#### 2.1 确保描述逻辑连贯

**问题**：描述顺序混乱，缺乏逻辑

**优化方法**：
1. 按照"环境→人物→动作→光线→氛围"的顺序描述
2. 确保因果关系清晰
3. 删除突兀的跳转

**示例**：

**优化前**：
```
傍晚昏黄的街灯，透过薄雾洒在粗糙的路面上，形成斑驳的光影。历史纪录片风格，中景镜头，1947年台北太平町天马茶房前。40岁的林江迈，身穿打补丁的粗布衣裳，脸上写满疲惫，蹲在路边摆摊。
```

**优化后**：
```
历史纪录片风格，中景镜头，1947年台北太平町天马茶房前。40岁的林江迈，身穿打补丁的粗布衣裳，脸上写满疲惫，蹲在路边摆摊。傍晚昏黄的街灯，透过薄雾洒在粗糙的路面上，形成斑驳的光影。
```

#### 2.2 优化句子结构

**问题**：句子过长或过短，影响阅读流畅性

**优化方法**：
1. 将过长的句子拆分为2-3个短句
2. 将过短的句子合并
3. 使用恰当的连接词

**示例**：

**优化前**：
```
历史纪录片风格，中景镜头，1947年台北太平町天马茶房前，一位40岁的台湾寡妇，身穿打补丁的粗布衣裳，脸上写满疲惫，眼神中有对明天的担忧，蹲在路边摆摊，身体前倾，正在整理香烟摊位。
```

**优化后**：
```
历史纪录片风格，中景镜头，1947年台北太平町天马茶房前。40岁的林江迈，身穿打补丁的粗布衣裳，脸上写满疲惫，蹲在路边摆摊，正在整理香烟摊位。
```

#### 2.3 增强画面感

**问题**：描述过于抽象，缺乏画面感

**优化方法**：
1. 使用具体的视觉描述
2. 避免抽象概念
3. 使用感官语言

**示例**：

**优化前**：
```
40岁的林江迈，表现得很疲惫，看起来很辛苦，蹲在路边摆摊。
```

**优化后**：
```
40岁的林江迈，脸上写满疲惫，眼神中有对明天的担忧，蹲在路边摆摊，身体前倾。
```

### 规则3：技术参数优化

#### 3.1 确保摄影术语准确

**问题**：摄影术语使用错误或不恰当

**优化方法**：
1. 检查摄影术语的准确性
2. 使用标准的摄影术语
3. 删除不恰当的术语

**示例**：

**优化前**：
```
广角镜头，特写镜头，微距镜头，光圈 f/2.8，ISO 400。
```

**优化后**：
```
中景镜头，光圈 f/2.8，ISO 400，胶片质感。
```

**常见摄影术语**：
- 景别：远景、全景、中景、中近景、近景、特写
- 构图：三分法、对称构图、黄金分割
- 光线：侧光、逆光、顺光、顶光
- 色调：冷色调、暖色调、中性色调

#### 3.2 调整画面参数

**问题**：画面参数不合理或不一致

**优化方法**：
1. 根据场景调整光圈（景深需求）
2. 根据光线条件调整ISO
3. 统一质感描述

**示例**：

**优化前**：
```
光圈 f/1.4，ISO 100，数字质感。
```

**优化后**：
```
光圈 f/2.8，ISO 400，胶片质感。
```

**画面参数建议**：
- 历史纪录片风格：光圈 f/2.8-f/5.6，ISO 400-800，胶片质感
- 电影感风格：光圈 f/1.4-f/2.8，ISO 200-400，电影质感
- 纪实摄影风格：光圈 f/5.6-f/11，ISO 800-1600，纪实质感

#### 3.3 统一风格参数

**问题**：风格参数不统一，影响整体一致性

**优化方法**：
1. 确定整体风格
2. 统一风格参数
3. 删除冲突的风格描述

**示例**：

**优化前**：
```
历史纪录片风格，电影感风格，纪实摄影风格。
```

**优化后**：
```
历史纪录片风格。
```

### 规则4：历史准确性优化

#### 4.1 修正历史错误

**问题**：时代特征、文化背景、地理位置错误

**优化方法**：
1. 对比特征库验证时代特征
2. 查阅历史资料确保准确性
3. 修正历史错误

**示例**：

**优化前**：
```
1947年林江迈，身穿现代西装，站在玻璃幕墙摩天大楼前。
```

**优化后**：
```
1947年林江迈，身穿打补丁的粗布衣裳，站在木结构建筑前。
```

#### 4.2 补充历史细节

**问题**：历史细节不足，缺乏真实感

**优化方法**：
1. 根据时代特征补充细节
2. 参考历史资料
3. 使用特征库的`prompt_snippet`

**示例**：

**优化前**：
```
1947年林江迈，身穿粗布衣裳，蹲在路边摆摊。
```

**优化后**：
```
1947年林江迈，身穿打补丁的粗布衣裳，蹲在路边摆摊，身体前倾，正在整理香烟摊位。
```

#### 4.3 确保文化尊重

**问题**：文化描述不准确或不尊重

**优化方法**：
1. 了解文化背景
2. 使用准确的术语
3. 避免刻板印象

**示例**：

**优化前**：
```
明代大臣鞠躬朝拜皇帝。
```

**优化后**：
```
明代大臣跪地朝拜皇帝。
```

## 优化流程

### 步骤1：简洁性优化

1. 删除冗余描述
2. 合并相似元素
3. 保留核心视觉特征

### 步骤2：流畅性优化

1. 确保描述逻辑连贯
2. 优化句子结构
3. 增强画面感

### 步骤3：专业性优化

1. 确保摄影术语准确
2. 调整画面参数
3. 统一风格参数

### 步骤4：准确性优化

1. 修正历史错误
2. 补充历史细节
3. 确保文化尊重

### 步骤5：最终检查

1. 检查prompt长度（建议≤200字）
2. 检查核心特征完整性
3. 检查技术参数合理性

## 优化示例

### 示例1：Beat Board Prompt优化

**优化前**（150字）：
```
历史纪录片风格，中景镜头，1947年台北太平町天马茶房前。一位40岁的台湾寡妇，身穿打补丁的粗布衣裳，脸上写满疲惫，眼神中有对明天的担忧，蹲在路边摆摊，身体前倾，正在整理香烟摊位。傍晚昏黄的街灯，透过薄雾洒在粗糙的路面上，形成斑驳的光影。灰褐色和旧黄色的色调，破旧的木结构建筑和日式装饰，拥挤陈旧的生活气息，紧张混乱的氛围。光圈 f/2.8，ISO 400，胶片质感。
```

**优化后**（120字）：
```
历史纪录片风格，中景镜头，1947年台北太平町天马茶房前。40岁的林江迈，身穿打补丁的粗布衣裳，脸上写满疲惫，蹲在路边摆摊，正在整理香烟摊位。傍晚昏黄的街灯透过薄雾洒在粗糙路面上，形成斑驳光影。灰褐色和旧黄色的色调，破旧木结构建筑和日式装饰，紧张混乱的氛围。光圈 f/2.8，ISO 400，胶片质感。
```

**优化要点**：
- 删除冗余："一位台湾寡妇"→"林江迈"
- 合并相似元素："眼神中有对明天的担忧"→删除（核心表情已包含）
- 简化描述："透过薄雾洒在粗糙的路面上"→"透过薄雾洒在粗糙路面上"
- 保留核心：人物、动作、环境、光线、氛围、参数

**优化后**（130字）：

## 常见问题和解决方案

### Q1：如何平衡简洁性和完整性？

A：优先保留核心视觉特征（人物、动作、环境），删除次要细节（背景人物、远距离元素）。确保prompt长度在200字以内。

### Q2：如何处理复杂的场景描述？

A：使用模板6（综合Prompt模板），详细描述核心元素，简化次要元素。如果仍然过长，考虑拆分为多个shot。

### Q3：如何确保摄影术语准确？

A：参考标准的摄影术语表，使用常见的摄影术语（景别、构图、光线、色调）。避免使用不常见的专业术语。

### Q4：如何处理历史准确性问题？

A：查阅特征库和历史资料，确保时代特征、文化背景、地理位置准确。如果不确定，使用通用的历史描述。
