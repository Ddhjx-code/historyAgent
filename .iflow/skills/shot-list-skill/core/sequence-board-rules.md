# Sequence Board生成规则

本文档定义大模型生成Sequence Board（四宫格/可变格）时应遵循的规则。

## Sequence Board目的

Sequence Board（四宫格/可变格展开板）用于：
1. **展开段落内部**：围绕Beat Board格子展开连续动作
2. **保持一致性**：继承Beat Board的人物/场景/光色
3. **确保连贯性**：检查屏幕方向、轴线、跳接规则

## 结构规则

### 默认结构（四格）

| 格子 | 类型 | 功能 |
|------|------|------|
| 1 | 起 | 建立局部空间与动机/威胁 |
| 2 | 承 | 行动推进（动作开始） |
| 3 | 转 | 冲突/受阻/反转（代价出现） |
| 4 | 合 | 结果落点 + 下一步钩子 |

### 动作戏结构（可替换）

| 格子 | 类型 | 功能 |
|------|------|------|
| 1 | 预备 | 准备动作 |
| 2 | 出手 | 动作开始 |
| 3 | 命中/受击 | 撞击瞬间 |
| 4 | 反应/余势 | 结果反应 |

### 格数选择

| 复杂度 | 格数 | 说明 |
|--------|------|------|
| 简单信息 | 2格 | 信息交代、场景转换 |
| 常规段落 | 4格 | 默认结构 |
| 动作复杂 | 6格 | 必须保证动作连续与空间可读 |

## 连贯性规则

### 屏幕方向一致

**规则**：面向、运动方向、进出画方向应保持稳定

**检查项**：
- [ ] 角色在前一镜头朝左移动，下一镜头不能突然反向朝右
- [ ] 角色从画面右侧退出，下一镜头应从画面左侧入镜

### 动作中段切

**规则**：优先在动作明显运动中段/高峰切镜头

**检查项**：
- [ ] 避免在动作未起未落的尴尬瞬间切换
- [ ] 动作中段切让观众大脑自然将两个镜头的动作连接起来

**示例**：
- ✅ 挥拳动作在拳头到达最高点时切换
- ❌ 在拳头收回后切换

### 轴线管理

**规则**：对话/对峙先建立轴线与左右关系，机位尽量保持轴线同侧

**检查项**：
- [ ] 已建立轴线：机位保持同侧
- [ ] 必须越轴时，使用过渡手段（角色移动、中性过渡镜头）

### 避免跳接

**规则**：相邻镜头如果变化太小，容易产生跳接抖动感

**检查项**：
- [ ] 30度法则：切换镜头时机位至少偏移30度或推远/拉近适当距离
- [ ] 景别变化：远↔近，避免相同景别直接切换

## 连贯性检查报告

### 检查项

| 检查项 | 说明 | 状态 |
|--------|------|------|
| 屏幕方向 | 面向、运动方向一致 | passed/warning/failed |
| 动作连贯性 | 动作阶段连续（start→middle→climax→aftermath） | passed/warning/failed |
| 轴线管理 | 轴线稳定或使用过渡 | passed/warning/failed |
| 跳接风险 | 角度/景别变化足够 | passed/warning/failed |

### 问题描述

| 镜头ID | 问题类型 | 描述 | 严重程度 |
|--------|---------|------|---------|
| shot_001 | screen_direction | 屏幕方向不一致 | low/medium/high |

## 验收标准

生成Sequence Board后必须检查：

- [ ] **动作与因果**：起手—推进—冲突—落点是否清楚？
- [ ] **动作与因果**：动作是否连续、空间是否可读？
- [ ] **一致性**：是否继承Beat Board对应格的人物/场景/光色？
- [ ] **一致性**：变化是否有明确原因说明？
- [ ] **连贯性**：屏幕方向/视线/轴线是否稳定？
- [ ] **连贯性**：越轴是否有过渡手段？
- [ ] **剪接风险**：是否存在跳接风险（角度变化不足、景别变化不足）？
- [ ] **剪接风险**：是否使用动作中段切来提高衔接自然度？
- [ ] **节奏与张力**：段落内部是否有张弛与呼吸？
- [ ] **节奏与张力**：是否有信息控制（藏与露）与关键细节强调？
- [ ] **承接性**：最后一格是否能自然接到下一锚点/下一场景？

## 注意事项

1. **继承一致性**：必须继承Beat Board的人物/场景/光色
2. **动作连贯**：确保动作阶段连续（start→middle→climax→aftermath）
3. **避免跳接**：相邻镜头的角度/景别变化要足够
4. **节奏控制**：高强度段落后安排呼吸点（静镜/空镜）
- [ ] **画面要素**：每个镜头的景别、主体、环境、光线、色调清晰
- [ ] **叙事目的**：每个镜头的目的明确（信息/情绪/悬念/转折）
