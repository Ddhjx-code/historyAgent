# 节拍拆解规则

本文档定义大模型拆解节拍时应遵循的规则。

## 节拍强度分类

### 高强度（HIGH）

**判断标准**：
- 包含冲突、暴力关键词
- 包含转折关键词
- 包含强烈情绪关键词

**关键词**：
- 冲突：冲突、殴打、击打、暴力、镇压、枪击、死、杀、伤、爆炸、火烧、倒塌
- 转折：但、然而、突然、不料、竟然、谁知、却、反而
- 情绪：愤怒、悲伤、震惊、痛苦、绝望、恐惧、暴怒、悲痛欲绝

**示例**：
```
查缉员用枪托殴打这位可怜的寡妇。 → HIGH
但令人悲哀的是，台湾行政长官陈仪下令开枪镇压。 → HIGH
```

### 中强度（MEDIUM）

**判断标准**：
- 包含推进关键词
- 包含中度情绪变化

**关键词**：
- 推进：开始、继续、随后、接着、于是、因此、所以、导致
- 中度情绪：担忧、不安、期待、失望

**示例**：
```
消息迅速传开，民众聚集抗议。 → MEDIUM
这进一步激怒了全岛的民众。 → MEDIUM
```

### 低强度（LOW）

**判断标准**：
- 包含平静叙述关键词
- 主要为信息交代

**关键词**：
- 平静：是、在、位于、名叫、叫做、来自、生活在

**示例**：
```
事件的主人公，是一位名叫林江迈的40岁寡妇。 → LOW
为了养活这两个孩子，林江迈在街边摆起了烟纸摊。 → LOW
```

## 锚点选择规则

### 锚点类型与选择标准

| 锚点类型 | 选择标准 | 对应阶段 |
|---------|---------|---------|
| 世界观/基调建立 | 设定时代背景、整体氛围 | 开场 |
| 主角现状与目标/缺口 | 介绍主要角色及其处境 | 开场-发展 |
| 诱因事件 | 推动剧情发展的初始事件 | 发展 |
| 第一次转折 | 故事发生重要变化 | 发展 |
| 中点升级/认知反转 | 故事中期的重大转折 | 中段 |
| 危机/低谷 | 角色面临的最大挑战 | 中段-高潮 |
| 高潮前集结 | 为最终对决做准备 | 高潮前 |
| 高潮对抗/结果 | 故事的高潮和结果 | 高潮 |
| 结局/余韵 | 故事的结尾和反思 | 结局 |

### 选择步骤

1. **强度优先**：优先选择高强度节拍
2. **覆盖叙事弧**：确保覆盖所有阶段
3. **避免重复**：不要选择表达相同内容的节拍
4. **类型分配**：每种锚点类型至少一个

### 锚点分配示例

**事件：1947年二二八事件**

| 序号 | 节拍描述 | 强度 | 锚点类型 |
|------|---------|------|---------|
| 1 | 林江迈摆烟纸摊 | LOW | 主角现状与目标/缺口 |
| 2 | 查缉员出现 | MEDIUM | 诱因事件 |
| 3 | 枪托殴打林江迈 | HIGH | 第一次转折 |
| 4 | 陈文溪被误伤身亡 | HIGH | 危机/低谷 |
| 5 | 民众对政府失望 | MEDIUM | 中点升级/认知反转 |
| 6 | 陈仪下令开枪镇压 | HIGH | 高潮对抗/结果 |
| 7 | 全台湾爆发大规模反抗 | HIGH | 结局/余韵 |
| 8 | （补充节拍） | MEDIUM | - |
| 9 | （补充节拍） | MEDIUM | - |

## 节拍拆解表生成

### 表格格式

```markdown
| 序号 | 节拍描述 | 观众收获 | 强度 | 是否锚点 | 锚点类型 | 情绪 |
|------|---------|---------|------|---------|---------|------|
| 1 | {发生了什么} | {信息/情绪/悬念} | 低/中/高 | 是/否 | {锚点类型} | {情绪} |
```

### 统计信息

1. **节拍强度分布**：
   - 高强度节拍：X个（占总数X%）
   - 中强度节拍：X个（占总数X%）
   - 低强度节拍：X个（占总数X%）

2. **节奏分析**：描述节拍的节奏变化

3. **情绪曲线**：按顺序展示情绪变化

## 情绪曲线分析

### 情绪变化描述

| 节拍序号 | 情绪类型 | 情绪描述 | 节拍强度 |
|---------|---------|---------|---------|
| 1 | SOLEMN | 庄重肃穆 | 低 |
| 2 | TENSE | 紧张激烈 | 高 |
| 3 | DRAMATIC | 戏剧性张力 | 高 |

### 分析要点

1. **识别转折点**：情绪发生明显变化的位置
2. **节奏控制**：高强度节拍后应有呼吸点
3. **情绪连贯**：相邻节拍的情绪应有逻辑联系

## 验收标准

生成节拍拆解表后必须检查：

- [ ] **完整性**：是否覆盖完整叙事弧线？
- [ ] **完整性**：是否遗漏关键转折点？
- [ ] **清晰性**：每个节拍的叙事目的是否明确？
- [ ] **清晰性**：节拍强度标注是否合理？
- [ ] **选点**：选取的9个锚点是否覆盖关键拐点类型？
- [ ] **覆盖**：是否存在"重复信息点"的锚点（冗余）？

## 注意事项

1. **保持原始内容**：节拍描述应保留口播稿原始内容
2. **准确标注强度**：根据关键词准确判断强度
3. **合理选择锚点**：确保覆盖叙事弧线的关键拐点
4. **避免重复选择**：不要选择表达相同内容的节拍作为锚点
