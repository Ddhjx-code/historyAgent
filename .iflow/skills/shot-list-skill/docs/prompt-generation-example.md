# Prompt生成示例文档

## 概述

本文档展示从口播稿到最终Prompt的完整生成流程，包含脚本解析、节拍拆解、特征库查询、Beat Board生成、Sequence Board生成、一致性检查和Prompt优化。

## 示例选择

选择事例稿中的**事件一：公元138年2月25日 - 罗马帝国的权力交接**作为示例。

## 阶段1：脚本解析

### 输入：口播稿片段


## 阶段6：Prompt优化

### 优化前后对比

**优化前**（镜头1）：
```
罗马帝国宫殿的大理石柱廊，晴朗的天空，清晨的阳光。哈德良皇帝从宫殿深处走来，身穿紫色镶边托加长袍，威严的脸上带着深思熟虑的表情，头上戴着橄榄枝冠冕。平静而庄严，权力的气息弥漫在空气中。晨光透过高窗射入，在地面形成明亮的几何图案，背景是金色的马赛克地板和庄严的雕像，视觉焦点在皇帝的面部。
```

**优化后**（镜头1）：
```
罗马帝国宫殿大理石柱廊，晴朗天空，清晨阳光。哈德良皇帝从宫殿深处走来，身穿紫色镶边托加长袍，威严脸上带着深思熟虑表情，头戴橄榄枝冠冕。平静庄严，权力气息弥漫空气中。晨光透过高窗射入，地面形成明亮几何图案，背景是金色马赛克地板和庄严雕像，视觉焦点在皇帝面部。
```

**优化要点**：
- 删除冗余："的"字多次出现
- 简化描述："带着深思熟虑的表情"→"带着深思熟虑表情"
- 保留核心：人物、动作、环境、光线、氛围

## 常见问题

### Q1：如何处理多个人物的场景？

A：在Prompt中分别描述每个人物，使用"主体A...，主体B..."的结构。例如："哈德良皇帝坐在王座上，安敦宁·毕尤跪地朝拜。"

### Q2：如何处理复杂的建筑场景？

A：使用详细的建筑描述，参考特征库中的建筑特征。例如："大理石柱廊、穹顶、马赛克地板、高窗"。

### Q3：如何处理情绪变化？

A：在Sequence Board中，通过光线、表情、氛围的变化体现情绪。例如："哈德良皇帝神情突然变得严肃"。

### Q4：如何确保历史准确性？

A：严格使用特征库中的历史特征，参考历史资料，确保时代特征、文化背景、地理位置准确。
