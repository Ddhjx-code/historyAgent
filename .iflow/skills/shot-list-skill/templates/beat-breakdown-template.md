## 使用说明

### 节拍定义

每个节拍是一个最小叙事单元，必须回答：
- **发生了什么**：具体的动作、事件、变化
- **观众得到什么**：信息（背景、事实）、情绪（愤怒、悲伤、希望）、悬念（未解之谜）

### 强度标注

- **低强度**：平静的叙述、信息交代、环境描写
- **中强度**：轻微冲突、情绪变化、情节推进
- **高强度**：激烈冲突、重大转折、情绪爆发

### 锚点选择原则

1. **覆盖完整叙事弧线**：从开头到结尾
2. **覆盖关键转折点**：所有重要的情节变化
3. **避免重复信息点**：不要选择表达相同内容的节拍
4. **保持视觉连贯性**：锚点之间的视觉变化要合理

### 情绪类型

| 情绪类型 | 适用场景 | 对应情绪 |
|---------|---------|---------|
| SOLEMN | 庄重历史、重大事件 | 庄重、肃穆、沉重 |
| TENSE | 冲突、对抗、危机 | 紧张、愤怒、恐惧 |
| HOPEFUL | 希望、成功、转折 | 充满希望、振奋 |
| SOMBER | 悲伤、失败、压抑 | 忧郁、悲凉、压抑 |
| DRAMATIC | 戏剧性高潮 | 戏剧性、张力、爆发 |
| NEUTRAL | 中性叙述 | 中性、平静 |
| PEACEFUL | 和平、安宁 | 宁静、祥和 |
| ANGRY | 愤怒、仇恨 | 愤怒、仇恨、仇恨 |
