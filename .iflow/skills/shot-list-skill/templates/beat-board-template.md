## Beat Board用途

### 1. 视觉锚点确立

Beat Board的9个格子作为后续Sequence Board的视觉锚点，确保：
- 人物外观在所有镜头中保持一致
- 场景环境在所有镜头中保持一致
- 光色基调在所有镜头中保持一致

### 2. 参考图片（垫图）

每个Beat Board格子生成后，作为对应Sequence Board组的参考图片：

```python
# 示例：格子0对应sequence_group_01
beat_board_00.png → sequence_group_01的所有镜头的reference_image
```

### 3. 风格统一

Beat Board确立的全局风格预设，所有Sequence Board必须继承：
- visual_style
- color_grading
- lighting_type
- mood

## 提示词生成规则

### 基础结构

遵循gemini-image-prompt-guide.md的最佳实践：

```
[景别]，[环境]。
[主体描述]，[动作/状态]。
[光线描述]，[光影效果]。
[角度/构图说明]。
[氛围/情绪]。
```

### 模块化组合

使用intelligent-prompt-generator的模块化系统：

1. **构图/景别模块**: 决定取景范围
2. **机位/角度模块**: 决定视角
3. **镜头效果模块**: 决定空间感
4. **光影氛围模块**: 决定情绪基调
5. **主体描述模块**: 决定画面内容
6. **动作/姿态模块**: 决定瞬间状态
7. **风格模块**: 决定美术风格

### 方法论规则应用

应用方法论补充.md的规则库：

- **叙事清晰规则**: 每格必须回答"拍什么、怎么拍、为什么"
- **构图与视觉引导规则**: 单帧必须成立，视觉引导明确
- **光影与色彩规则**: 光质、光源方向、时间色温、色彩情绪
- **镜头语言规则**: 景别功能、角度心理含义、镜头效果
