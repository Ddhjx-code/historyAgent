# Prompt模板库

## 概述

本文档提供了6种标准Prompt模板，用于生成高质量的图像prompt。这些模板遵循叙事描述式风格，确保历史准确性和视觉一致性。

## 模板1：Beat Board Prompt模板

**用途**：生成9个锚点的视觉描述，建立人物/场景/光线的一致性基础

**结构**：5个部分

```
[第1部分：风格、景别、环境]
[第2部分：主体、动作]
[第3部分：光线、光影]
[第4部分：色调、氛围]
[第5部分：画面参数]
```

**模板**：

```
[风格预设]，[景别]，[时代背景] [地点描述]。
[主体描述]，[服饰特征]，[表情神态]，[动作姿态]。
[光线来源]，[光影效果]。
[色调描述]，[色彩情绪]，[氛围基调]。
[画面参数：光圈、ISO、质感等]。
```

**变量说明**：

| 变量 | 说明 | 示例 |
|------|------|------|
| [风格预设] | 图像风格 | 历史纪录片风格、电影感风格、纪实摄影风格 |
| [景别] | 镜头距离 | 远景、全景、中景、中近景、近景、特写 |
| [时代背景] | 历史时代 | 1947年、2世纪、7世纪、15世纪 |
| [地点描述] | 场景环境 | 台北太平町天马茶房前、罗马帝国宫殿 |
| [主体描述] | 人物/动物 | 40岁的林江迈、哈德良皇帝 |
| [服饰特征] | 服装描述 | 身穿打补丁的粗布衣裳、紫色镶边托加长袍 |
| [表情神态] | 面部表情 | 脸上写满疲惫、威严庄严、愤怒紧张 |
| [动作姿态] | 身体动作 | 蹲在路边摆摊、坐在王座上、向前奔跑 |
| [光线来源] | 光线来源 | 傍晚昏黄的街灯、高窗射入的自然光、烛光 |
| [光影效果] | 光影描述 | 形成斑驳的光影、柔和的漫射光、强烈的对比 |
| [色调描述] | 主色调 | 灰褐色和旧黄色、白色和金色、朱红色和黄色 |
| [色彩情绪] | 色彩情感 | 破旧陈旧、威严庄严、繁华宏大 |
| [氛围基调] | 整体氛围 | 紧张混乱、庄严神圣、繁华热闹 |
| [画面参数] | 技术参数 | 光圈 f/2.8，ISO 400，胶片质感 |

**示例**：

```
历史纪录片风格，中景，1947年台北太平町天马茶房前。
40岁的林江迈，身穿打补丁的粗布衣裳，脸上写满疲惫，眼神中有对明天的担忧，蹲在路边摆摊，身体前倾，正在整理香烟摊位。
傍晚昏黄的街灯，透过薄雾洒在粗糙的路面上，形成斑驳的光影。
灰褐色和旧黄色的色调，破旧的木结构建筑和日式招牌，拥挤陈旧的生活气息，紧张混乱的氛围。
光圈 f/2.8，ISO 400，胶片质感。
```

**使用场景**：
- 生成Beat Board的9个格子
- 建立视觉锚点
- 确保人物/场景/光线一致性

## 模板2：Sequence Board Prompt模板（起）

**用途**：生成动作开始的场景，引入环境和人物

**结构**：4个部分

```
[第1部分：场景引入]
[第2部分：人物出现]
[第3部分：情绪铺垫]
[第4部分：视觉细节]
```

**模板**：

```
[场景环境描述]，[天气条件]，[时间]。[人物名称] [出现在场景的方式]，[外貌特征]，[服饰描述]。[初始情绪]，[氛围描述]。[光线细节]，[背景元素]，[视觉焦点]。
```

**变量说明**：

| 变量 | 说明 | 示例 |
|------|------|------|
| [场景环境描述] | 场景环境 | 罗马帝国宫殿的大理石柱廊、长安城门的朱红色城墙 |
| [天气条件] | 天气 | 晴朗、阴天、薄雾、下雨 |
| [时间] | 时间 | 清晨、正午、傍晚、深夜 |
| [人物名称] | 人物名字 | 林江迈、哈德良皇帝、玄奘高僧 |
| [出现在场景的方式] | 出现方式 | 走进、站立、出现、从远处走来 |
| [外貌特征] | 外貌描述 | 40岁的台湾寡妇、威严的罗马皇帝、庄严的唐代高僧 |
| [服饰描述] | 服饰 | 打补丁的粗布衣裳、紫色镶边托加长袍、赭红色僧袍 |
| [初始情绪] | 初始情绪 | 平静、期待、警惕、疲惫 |
| [氛围描述] | 氛围 | 安静、紧张、庄严、热闹 |
| [光线细节] | 光线 | 晨光透过窗户、傍晚的街灯、烛光摇曳 |
| [背景元素] | 背景 | 柱廊、城墙、人群、建筑 |
| [视觉焦点] | 焦点 | 人物的面部、手中的物品、远处的目标 |

**示例**：

```
罗马帝国宫殿的大理石柱廊，晴朗的天空，清晨的阳光。哈德良皇帝从宫殿深处走来，身穿紫色镶边托加长袍，威严的脸上带着深思熟虑的表情，头上戴着橄榄枝冠冕。平静而庄严，权力的气息弥漫在空气中。晨光透过高窗射入，在地面形成明亮的几何图案，背景是金色的马赛克地板和庄严的雕像，视觉焦点在皇帝的面部。
```

**使用场景**：
- Sequence Board的第一个格子（起）
- 引入新的场景或人物
- 建立情绪基础

## 模板3：Sequence Board Prompt模板（承）

**用途**：生成动作发展的场景，推进情节

**结构**：4个部分

```
[第1部分：动作发展]
[第2部分：互动细节]
[第3部分：情绪变化]
[第4部分：视觉推进]
```

**模板**：

```
[主体A] [动作描述]，[主体B] [反应描述]。[互动细节]，[动作连续性]。[情绪升级/变化]，[氛围转变]。[镜头变化]，[视觉重点]，[背景变化]。
```

**变量说明**：

| 变量 | 说明 | 示例 |
|------|------|------|
| [主体A] | 主要人物 | 查缉员、皇帝、文官 |
| [动作描述] | 动作细节 | 快速向林江迈走来、坐在王座上、展开卷轴 |
| [主体B] | 对应人物 | 林江迈、大臣、僧人 |
| [反应描述] | 反应细节 | 惊慌失措、恭敬地低头、仔细聆听 |
| [互动细节] | 互动方式 | 走到面前、对视、对话 |
| [动作连续性] | 动作连贯 | 动作流畅自然、没有跳接 |
| [情绪升级/变化] | 情绪变化 | 从平静到紧张、从尊敬到担忧 |
| [氛围转变] | 氛围变化 | 气氛逐渐紧张、气氛变得更加庄严 |
| [镜头变化] | 镜头运动 | 镜头逐渐推近、镜头缓慢平移 |
| [视觉重点] | 焦点变化 | 聚焦在两人的表情、聚焦在卷轴的内容 |
| [背景变化] | 背景改变 | 背景人群逐渐增多、背景光线逐渐变暗 |

**示例**：

```
专卖局查缉员快步向林江迈走去，林江迈惊慌失措，试图保护自己的货物。查缉员走到林江迈面前，粗暴地翻检她的香烟摊位，两人的动作流畅自然。气氛从平静逐渐变得紧张，周围群众开始注意到这边的情况。镜头逐渐推近，聚焦在两人的表情上，林江迈脸上写满担忧，查缉员表情严肃冷漠，背景中围观群众逐渐增多。
```

**使用场景**：
- Sequence Board的第二个格子（承）
- 推进情节发展
- 建立冲突或互动

## 模板4：Sequence Board Prompt模板（转）

**用途**：生成冲突转折的场景，情绪高潮

**结构**：4个部分

```
[第1部分：冲突爆发]
[第2部分：强烈动作]
[第3部分：情绪高潮]
[第4部分：视觉冲击]
```

**模板**：

```
[冲突发生]，[关键动作]。[强烈动作描述]，[激烈互动]。[情绪爆发]，[极度紧张/悲伤/愤怒]。[特写细节]，[视觉冲击]，[戏剧性光影]。
```

**变量说明**：

| 变量 | 说明 | 示例 |
|------|------|------|
| [冲突发生] | 冲突事件 | 查缉员推搡林江迈、大臣跪地求情、僧人展开佛经 |
| [关键动作] | 关键动作 | 林江迈失去平衡、皇帝拍案而起、僧人诵经 |
| [强烈动作描述] | 动作细节 | 粗暴地推搡、愤怒地挥动手臂、庄严地举起佛经 |
| [激烈互动] | 互动细节 | 两人推搡、众人跪拜、僧人与皇帝对话 |
| [情绪爆发] | 情绪爆发 | 愤怒爆发、悲伤涌上心头、庄严神圣 |
| [极度紧张/悲伤/愤怒] | 情绪强度 | 极度紧张、极度悲伤、极度愤怒 |
| [特写细节] | 特写镜头 | 林江迈倒地受伤的表情、皇帝愤怒的眼神、僧人庄严的面容 |
| [视觉冲击] | 视觉效果 | 鲜血渗出、金光闪烁、佛光普照 |
| [戏剧性光影] | 光影效果 | 强烈的对比光、逆光、剪影 |

**示例**：

```
查缉员突然推搡林江迈，她失去平衡倒在地上，额头撞到粗糙的路面，鲜血渗出。查缉员粗暴地拉扯林江迈的衣领，两人发生激烈的肢体冲突。愤怒的情绪在人群中爆发，群众开始聚集，声音逐渐高涨。特写镜头捕捉林江迈痛苦的面部表情，额头上的鲜血清晰可见，强烈的对比光在伤口处形成戏剧性的效果，视觉冲击强烈。
```

**使用场景**：
- Sequence Board的第三个格子（转）
- 冲突或转折点
- 情绪高潮

## 模板5：Sequence Board Prompt模板（合）

**用途**：生成动作结束的场景，收束场景

**结构**：4个部分

```
[第1部分：动作结束]
[第2部分：最终状态]
[第3部分：情绪沉淀]
[第4部分：视觉收束]
```

**模板**：

```
[动作结果]，[最终状态]。[人物状态]，[场景变化]。[情绪沉淀]，[余韵]。[镜头拉远]，[全貌展示]，[场景收束]。
```

**变量说明**：

| 变量 | 说明 | 示例 |
|------|------|------|
| [动作结果] | 动作结果 | 林江迈倒在地上、皇帝宣布决定、僧人结束讲经 |
| [最终状态] | 最终状态 | 痛苦呻吟、庄严肃立、安静祈祷 |
| [人物状态] | 人物状态 | 林江迈倒在路边、皇帝坐在王座上、僧人双手合十 |
| [场景变化] | 场景变化 | 人群聚集、大臣退下、信徒散去 |
| [情绪沉淀] | 情绪沉淀 | 愤怒的余韵、庄严的氛围、宁静的祥和 |
| [余韵] | 后续影响 | 冲突即将升级、帝国将继续繁荣、佛法将传承 |
| [镜头拉远] | 镜头运动 | 镜头缓慢拉远、镜头向上仰拍 |
| [全貌展示] | 全貌 | 展示整个混乱的现场、展示整个宫殿的宏伟、展示整个寺庙的庄严 |
| [场景收束] | 收束方式 | 画面逐渐变暗、画面定格、画面淡出 |

**示例**：

```
林江迈倒在地上痛苦呻吟，额头上的鲜血还在渗出，查缉员站在一旁冷漠观望。群众完全包围了他们，愤怒的声音此起彼伏，冲突即将升级到新的层面。愤怒的情绪在空气中弥漫，预示着更大的风暴即将来临。镜头缓慢拉远，展示整个混乱的现场，人群、建筑、昏黄的街灯，画面逐渐变暗，定格在充满张力的瞬间。
```

**使用场景**：
- Sequence Board的第四个格子（合）
- 场景结束
- 建立过渡到下一场景

## 模板使用指南

### 选择模板

1. **Beat Board**：使用模板1
2. **Sequence Board起**：使用模板2
3. **Sequence Board承**：使用模板3
4. **Sequence Board转**：使用模板4
5. **Sequence Board合**：使用模板5

## 常见问题

### Q1：如何处理模板中不需要的变量？

A：删除不需要的变量，保留核心元素。例如，如果场景中没有次要人物，可以删除次要人物相关的变量。

### Q2：如何处理多个人物的场景？

A：在[主要人物]和[次要人物]中分别描述，使用[互动关系]说明人物之间的互动。

### Q3：如何处理复杂的建筑场景？

A：使用模板6，详细描述建筑特征、环境元素和背景元素。

### Q4：如何处理情绪变化的场景？

A：使用模板3（承）和模板4（转），重点描述[情绪升级/变化]和[情绪爆发]。
