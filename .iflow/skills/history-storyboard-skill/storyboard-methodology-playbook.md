# 历史故事分镜方法论

## 基本原则

### 1. 历史准确性优先
- 所有视觉元素必须符合历史时期的真实情况
- 服饰、建筑、道具必须经过历史考证
- 人物的言行举止必须符合当时的社会礼仪
- 场景设计必须反映真实的历史地理环境

### 2. 视觉叙事清晰
- 每个镜头都应该有明确的叙事目的
- 观众能够从画面中理解故事进展
- 避免过于抽象或难以理解的视觉表达

### 3. 人物与场景一致性
- 同一人物在不同场景中的外貌特征保持一致
- 同一场景在不同镜头中的环境特征保持一致
- 光影效果在不同镜头中保持逻辑连贯

### 4. 画面连贯可剪辑
- 相邻镜头之间有合理的视觉衔接
- 镜头运动符合观众的视觉预期
- 剪辑点选择符合叙事节奏

## 节拍拆解规则

### 识别叙事曲线的关键拐点
1. **开场/背景建立**：设定时代背景、环境氛围
2. **人物登场**：介绍主要人物及其特征
3. **冲突萌芽**：展示故事的核心矛盾或事件
4. **关系建立**：展现人物之间的关系网络
5. **转折点**：故事的关键转折时刻
6. **情感升华**：情感达到高点的时刻
7. **主动抉择**：主角做出决定性选择
8. **成长转变**：角色完成转变的过程
9. **结局/意义**：故事的收束和历史意义

### 历史背景融入
- 每个节拍都要考虑其历史背景
- 描述历史事件时要准确反映其影响
- 人物行为要符合当时的社会环境和历史条件

## Beat Board 九宫格规则

### 九宫格布局原则
- **[1,1] 开场/背景**：建立历史时期和时代特征
- **[1,2] 人物登场**：展现主角身份和性格
- **[1,3] 冲突萌芽**：展示核心历史事件或矛盾
- **[2,1] 关系建立**：展现人物关系和社会结构
- **[2,2] 转折点**：关键历史节点或决断时刻
- **[2,3] 情感升华**：深刻情感表达
- **[3,1] 主动抉择**：历史人物的决策过程
- **[3,2] 成长转变**：角色的成长轨迹
- **[3,3] 结局/意义**：历史影响和遗产

### 历史细节要求
- **服饰**：准确反映历史时期的款式、材质、色彩
- **建筑**：如实反映当时的建筑风格和工艺
- **道具**：使用的器物、武器、工具等要有历史依据
- **礼仪**：人物的言行举止符合当时的礼仪规范
- **地理**：展现真实的历史地理环境

## Sequence Board 四宫格规则

### 继承性原则
- 四宫格必须继承九宫格对应格的人物/场景/光色描述
- 发生变化必须显式说明原因
- 保持人物和场景的视觉一致性

### 连贯性要求
- 相邻帧之间有合理的视觉衔接
- 动作轨迹符合物理规律
- 光影效果保持逻辑连贯

### 节奏控制
- **铺垫期**：较长镜头，慢节奏
- **发展期**：加快节奏，增加切换
- **高潮期**：快速剪辑，动态镜头
- **舒缓期**：慢镜头，静态画面
- **结局期**：回顾性镜头，稳定节奏

## 历史考证指南

### 服饰考证
- 查阅历史文献和考古资料
- 参考同时期的绘画、雕塑、壁画
- 注意不同阶层、不同身份的服饰差异
- 考虑地域和民族特色

### 建筑考证
- 研究当时的建筑风格和工艺
- 参考现存的古建筑遗址
- 注意不同时期的建筑变化
- 考虑地理环境和气候影响

### 道具考证
- 确认器物的时代特征
- 参考考古出土文物
- 注意不同地区的器物差异
- 考虑使用场景的合理性

### 礼仪考证
- 研究当时的社会礼仪规范
- 注意不同场合的礼仪要求
- 考虑不同阶层的礼仪差异
- 尊重历史文化的真实性

- [ ] 人物行为符合当时的社会礼仪
- [ ] 场景反映真实的历史地理环境

- [ ] 每个镜头有明确的叙事目的
- [ ] 画面清晰易懂
- [ ] 叙事节奏合理
- [ ] 情感表达准确

### 一致性
- [ ] 人物特征保持一致
- [ ] 场景特征保持一致
- [ ] 光影效果保持连贯
- [ ] 视觉风格统一

### 连贯性
- [ ] 相邻镜头有合理衔接
- [ ] 镜头运动符合视觉预期
- [ ] 剪辑点选择符合节奏
