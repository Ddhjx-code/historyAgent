# 动态提示词方法论

## 基本原则

### 1. 从分镜到动态的转化
- 动态提示词必须基于静态分镜的叙事目的与情绪
- 动作设计必须与分镜画面内容匹配
- 保持分镜中确立的人物与场景一致性

### 2. 镜头运动的逻辑性
- 镜头运动必须符合叙事目的
- 镜头运动必须符合观众的视觉预期
- 镜头运动必须符合物理规律

### 3. 动作的真实性
- 人物动作必须符合历史时期的身体特征
- 人物动作必须符合社会礼仪规范
- 动作轨迹必须符合物理规律

### 4. 节奏控制
- 通过镜头速度控制叙事节奏
- 通过动作幅度控制情感强度
- 通过场景切换控制紧张程度

## 从分镜到动态提示词的转化

### 关键帧动态提示词
**目的**：为九宫格每个格生成对应的动态提示词，作为该组四宫格的锚点

**转化步骤**：
1. 读取 beat-board-prompt.md 中对应格的描述
2. 读取 beat-breakdown.md 中该节拍的叙事目的与情绪
3. 设计镜头运动（静态或动态）
4. 设计人物动作（保持姿态或微妙动作）
5. 生成关键帧动态提示词

**示例**：
- **九宫格描述**："荆轲身披白衣，手持长剑，站在河边的芦苇丛中"
- **关键帧动态提示词**："镜头缓慢推进，聚焦在荆轲的面部表情。荆轲手持长剑，目光坚定，微风吹拂着他的白衣和芦苇。整体氛围肃杀而悲壮，寒风萧瑟，芦苇摇曳。"

### 四宫格展开动态提示词
**目的**：为每个四宫格帧生成对应的动态提示词，展开关键帧的段落动作

**转化步骤**：
1. 读取 sequence-board-prompt.md 中对应帧的描述
2. 继承对应关键帧的人物/场景/光色描述
3. 设计镜头运动
4. 设计人物动作和互动
5. 生成四宫格动态提示词

**示例**：
- **关键帧描述**："荆轲身披白衣，手持长剑，站在河边"
- **四宫格描述**："荆轲转身，向太子丹行礼"
- **四宫格动态提示词**："镜头跟随荆轲转身动作，展现他向太子丹行礼的庄重姿态。荆轲身披白衣，手持长剑，动作优雅而有力。太子丹面色凝重，向荆轲回礼。背景是易水河畔，芦苇在寒风中摇曳。"

## 镜头运动指南

### 基础镜头运动
- **推镜头（Dolly In）**：镜头向前推进，聚焦主体
- **拉镜头（Dolly Out）**：镜头向后拉远，展现环境
- **摇镜头（Pan）**：镜头左右或上下转动
- **移镜头（Truck）**：镜头平行移动
- **跟镜头（Follow）**：镜头跟随主体运动
- **升镜头（Pedestal Up）**：镜头向上移动
- **降镜头（Pedestal Down）**：镜头向下移动

### 组合镜头运动
- **推摇**：推进的同时摇动镜头
- **拉摇**：拉远的同时摇动镜头
- **环绕**：围绕主体旋转拍摄
- **升降推拉**：升降的同时推拉镜头

### 镜头运动选择原则
- **情感表达**：推进镜头增强情感，拉远镜头营造氛围
- **叙事重点**：跟镜头突出主体，环绕镜头展现关系
- **节奏控制**：快速运动增加紧张，慢速运动营造氛围

## 人物动作指南

### 基础动作类型
- **静态动作**：保持姿态，只有细微的呼吸或眨眼
- **微动作**：手势、转头、微笑等小幅度动作
- **常规动作**：行走、坐立、举手等日常动作
- **大幅度动作**：奔跑、跳跃、战斗等剧烈动作

### 历史人物动作特征
- **威严庄重**：君王、贵族的动作通常缓慢而有力
- **恭敬谦卑**：臣民、侍从的动作通常谦卑而恭敬
- **豪放不羁**：侠客、文人的动作通常自由而豪放
- **严谨规范**：武士、士兵的动作通常规范而严谨

### 动作真实性要求
- **身体特征**：考虑人物的年龄、性别、身体条件
- **服饰限制**：考虑服饰对动作的限制和影响
- **礼仪规范**：考虑社会礼仪对动作的要求
- **物理规律**：动作轨迹必须符合物理规律

## 环境渲染指南

### 光影效果
- **自然光**：阳光、月光、星光等自然光源
- **人造光**：火把、蜡烛、灯笼等人造光源
- **光影变化**：日出日落、云层移动等光影变化
- **光影氛围**：明暗对比、光影投射营造氛围

### 天气效果
- **晴天**：阳光明媚，视野清晰
- **阴天**：光线柔和，氛围沉郁
- **雨天**：雨水飘落，氛围悲凉
- **雪天**：雪花飘落，氛围纯洁
- **风天**：风势强劲，氛围肃杀

### 季节特征
- **春季**：万物复苏，生机勃勃
- **夏季**：炎热难耐，气氛紧张
- **秋季**：萧瑟凄凉，悲壮苍凉
- **冬季**：寒冷刺骨，严峻肃穆
