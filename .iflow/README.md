# 历史故事视频展示项目

## 项目简介

这是一个参考 manju 项目创建的历史故事视频展示项目，用于将历史故事转换成视觉叙事结构，生成视频故事板。

## 项目结构

```
history/
├── scripts/          # 历史故事脚本目录
│   └── 第一章-荆轲刺秦王.md
├── outputs/          # 生成的提示词输出目录
├── templates/        # 模板文件目录
│   ├── history-script-template.md    # 历史故事脚本模板
│   ├── beat-board-template.md        # Beat Board九宫格提示词模板
│   ├── motion-prompt-template.md     # Motion动态提示词模板
│   └── sequence-board-template.md    # Sequence序列板提示词模板
└── README.md         # 项目说明文档
```

## 模板说明

### 1. 历史故事脚本模板 (history-script-template.md)
用于编写历史故事脚本，包含：
- 基本信息（故事名称、历史时期、主要人物等）
- 场景描述（地点、时间、人物、环境、动作、对话等）
- 历史注释（真实历史背景和事件信息）
- 人物小传（人物身份、性格、历史作用等）
- 视觉风格建议（服装、场景、色调、镜头等）

### 2. Beat Board九宫格提示词模板 (beat-board-template.md)
用于生成九宫格视觉叙事锚点，包含：
- 基本信息
- 九宫格布局（开场/背景、人物登场、冲突萌芽、关系建立、转折点、情感升华、主动抉择、成长、结局/意义）
- 镜头设计说明
- 历史准确性要求

### 3. Motion动态提示词模板 (motion-prompt-template.md)
用于生成动态场景的详细描述，包含：
- 基本信息
- 场景描述（镜头运动、环境、人物、动作、氛围）
- 历史细节要求
- 镜头语言建议

### 4. Sequence序列板提示词模板 (sequence-board-template.md)
用于生成详细序列帧，包含：
- 基本信息
- 序列描述（时长、核心事件）
- 帧描述（画面、镜头、音效、字幕）
- 画面描述要求
- 历史准确性要求
- 节奏控制建议

## 使用方法

1. **编写历史故事脚本**
   - 在 `scripts/` 目录下创建新的脚本文件
   - 参考 `history-script-template.md` 模板编写内容

2. **生成提示词**
   - 基于脚本内容，使用对应的模板生成各种提示词
   - Beat Board：生成九宫格视觉叙事锚点
   - Motion：生成动态场景描述
   - Sequence：生成详细序列帧

3. **输出文件**
   - 将生成的提示词保存到 `outputs/` 目录

## 特色

- 专为历史故事设计，注重历史准确性
- 包含多种视觉叙事工具（Beat Board、Motion、Sequence）
- 提供详细的镜头设计和历史细节要求
- 支持多种输出格式（纪录片、电影、动画）

## 注意事项

- 所有模板都注重历史准确性，包括服饰、建筑、道具、礼仪等
- 使用前请了解相关历史背景和资料
- 建议参考真实历史文献和考古资料编写脚本
